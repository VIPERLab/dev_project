# 穷游 App

穷游 App iPhone 客户端，由原先的**穷游锦囊**升级而来。

https://itunes.apple.com/cn/app/id563467866?mt=8

## 功能

* 目的地（锦囊）
* 折扣
* 行程
* 身边人（聊天室）
* 个人中心

## 开发环境

* xcode 5+
* iOS 6+
* cocoapods

## 分支说明

* `master` - 线上发布的稳定版本（配合 tags ）
* `develop` - 开发版本

## 安装

1. 克隆项目代码

    $ git clone git@github.com:qyer-inc/qyer_app_iphone.git


2. 安装 Cocoapods 并更新项目依赖
  
    $ pod update

3. 打开 QYER.xcworkspace
